# .github

Default community health files and AI agent instructions for [yilmaz-games](https://github.com/yilmaz-games) repositories.

## What's here

- `.github/instructions/code-review.instructions.md` — Default code review rules for AI reviewers (Copilot, etc.). Repos can override by adding their own file at the same path.
- `.github/copilot-instructions.md` — Default behavior for the Copilot coding agent across all repos.

## How it works

GitHub automatically applies files from an org's `.github` repo as defaults for all repos that don't have their own copies. See [GitHub docs](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
